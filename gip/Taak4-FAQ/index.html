Tâche 4: FAQ ~ Youtube

Tue, Oct 20, 2015

Si vous avez des questions sur l’utilisation de YouTube, il y a une FAQ ci-dessous!


Foire Aux Questions

Les 4 questions que j’ai préparées avec Christophe Dumont:


Où puis-je trouver mes vidéos préférées?

Dans votre menu à gauche, cliquez sur la playlist “Favorites”.


Si vous faites une playlist sur un ordinateur, comment est-ce que je vois cette playlist sur une autre ordinateur?

Vous voyez la playlist quand vous connectez avec votre account de Google.


Comment puis-je supprimer toute une playlist?

Vous devez sélectionner la playlist que vous voulez supprimer, à droite vous pouvez selectionner l’option: “ Supprimer la playlist”.


Quand je sélectionne un vidéo, le vidéo ne marche pas. Comment est-ce que je le vois maintenant?

  • Pressez sur f5 sur votre clavier.
  • Redémarrez votre ordinateur.
  • Essayez avec un autre navigateur.
  • Mettez à jour votre navigateur.

Les 2 questions que j’ai préparées tout seul

Il y a encore d’autres sites comme youtube?

  • Vimeo.com
  • Dailymotion.com
  • Veoh.com

J’ai entendu qu’il est possible de gagner de l’argent sur youtube, comment ça marche?

  • Vous créez une chaïne sur youtube.
  • Vous commencez de faire un vidéo.
  • Vous téléchargez la vidéo que vous avez créé.
  • Vous activez les annonces.
  • Quand les visiteurs de votre vidéo, regardent les annonces, vous gagnez de l’argent.
