Review of a software tool in the training company: MS Visio

Sun, May 8, 2016



MS Visio is a tool created by Microsoft, but it is not in the office package, so if you want to use it, you will need to buy it separately. Another bad thing about it is the price, it is 399.00 euros for the standard version.

This software can come in very handy if you are planning on making a very detailed drawing of almost anything in your environment. It includes some standard libraries like “ Networking”, “office”, etc . But of course you can download your very own libraries from the internet or download your own!

In the review below, which is a little bit outdated by now, they mostly talked about the pros of Visio and when it comes to finding any flaws by now, I honestly wish you good luck. All the flaws mentioned are most likely fixed in the newer versions of MS Visio. Or at least plug-ins have been developed that you can download to upgrade your Visio even more. Maybe you think “But it is an expensive piece of software”. I can’t deny that, but when you can get this software for your work as an IT person, architect, or any other job that requires a bit of drawing your work will be done quicker and it would be editable at all times.

A couple of the assets of Visio that are mentioned are the new file format since 2013, which now gives you the oppurtunity to use Visio within Sharepoint, it is ideal for working in teams. Yet another pro is that you can select a basic starting theme for your drawing project. The third awesome feature they mentioned, which I have used myself is the commenting function. It’s just great when you want to add some information on the top of your drawing, which doesn’t have to be background information . That also is another great feature of this tool. It allows you to put extra information into any object .

When you get to the point you can actually start a project and you have decided to make it a team project, in the latest version of Visio there is a live update function, so whenever you change the data, your entire team will see that it has changed. Now I am talking about younger versions of Visio, maybe an idea for the future is to upgrade the programming aspect behind objects, maybe it will even be used for desktop applications then.

Of course there are other drawing softwares which are probally cheaper as well, but there will be no live support of the Microsoft Helpdesk whenever you need it. Suprised as I was, reading other drawing software specifications, I haven’t found a single thing that Visio didn’t offer the users yet that another software would give you. Comparing what you will get and what you are paying for it, I actually think it isn’t a bad purchase at all.
